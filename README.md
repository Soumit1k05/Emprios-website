# Emprios-website 🛡️

**EMPRIOS - Official Website Repository**
The official codebase for EMPRIOS (emprios.in), a modern, high-conversion affiliate marketing platform.

## 🚀 Quick Setup

1.  **Install All Components**:
    ```bash
    npm run install-all
    ```
2.  **Environment Variables**:
    Create a `.env` file in the `/be` directory using `.env.example` as a template.
3.  **Run Dev Server**:
    ```bash
    npm run dev
    ```

## 🛠 For Moderators & Developers

We welcome contributions! Please see our [CONTRIBUTING.md](./CONTRIBUTING.md) for guidelines on how to:
-   Update site content (bundling, pricing, etc.)
-   Run the project locally
-   Ensure your code is error-free before pushing to GitHub.

## 📦 Project Structure
-   `/fe`: React + Vite Frontend (Aesthetics, UI/UX)
-   `/be`: Node.js + Express Backend (API, Data Management)
